# 🚣 강 건너기 탐색 시각화

농부·늑대·양·양배추를 모두 강 오른쪽으로 옮기는 문제를 **DFS(깊이 우선 탐색)** 와 **BFS(너비 우선 탐색)** 으로 푸는 과정을 시각화한 교육용 도구입니다.

🔗 **[바로 실행하기](https://8laos.github.io/river-crossing-viz/)**

---

## 문제 설명

| 규칙 | 내용 |
|------|------|
| 이동 | 농부는 한 번에 혼자 또는 한 명만 태울 수 있다 |
| 제약 1 | 양과 늑대만 남으면 늑대가 양을 잡아먹는다 |
| 제약 2 | 양과 양배추만 남으면 양이 양배추를 먹는다 |
| 목표 | 농부·늑대·양·양배추 모두 오른쪽 강변으로 이동 |

---

## 기능

### 탐색 방법 탭 전환
- **DFS (깊이 우선 탐색)**: 가능한 이동을 먼저 회색으로 모두 펼친 뒤, 하나씩 깊이 파고들며 탐색
- **BFS (너비 우선 탐색)**: 현재 상태에서 가능한 이동을 모두 시도한 뒤 다음 단계로 이동

### 강 시각화
- 배가 강을 건너는 과정을 **부드럽게 애니메이션**으로 표현
- 농부는 배 왼쪽, 동반자는 배 오른쪽에 탑승
- 이동 중 나머지 캐릭터는 원래 강변에 유지

### 상태 공간 트리
| 색상 | 의미 |
|------|------|
| 회색 (미탐색) | DFS에서 아직 평가하지 않은 후보 이동 |
| 노랑 | 현재 탐색 중 |
| 초록 | 유효한 상태 (안전) |
| 빨강 | 무효 또는 이미 방문한 상태 |
| 어두운 회색 | 백트래킹 (DFS) |
| 파랑 | 최종 해 경로 |

---

## 사용 방법

1. 상단 탭에서 **DFS** 또는 **BFS** 선택
2. 속도 슬라이더로 탐색 속도 조절
3. **AI 자동 탐색** 버튼 클릭
4. 강 그림(왼쪽)과 상태 공간 트리(오른쪽)로 탐색 과정 확인
5. **초기화** 버튼으로 다시 시작

---

## DFS vs BFS 비교

| | DFS | BFS |
|--|-----|-----|
| 탐색 방식 | 한 경로를 끝까지 파고든 뒤 백트래킹 | 모든 경로를 레벨별로 동시에 탐색 |
| 최단 경로 보장 | ❌ | ✅ |
| 메모리 사용 | 적음 | 많음 |
| 특징 | 깊이 우선, 막히면 되돌아옴 | 너비 우선, 가까운 상태부터 탐색 |
